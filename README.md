### CSV & PapaParse
A CSV file is a plain-text file that contains all the dataset values in a table set format. These are the values to be plugged into script files. PapaParse is a file Library that reads CSV data from strings and local files and converts them to an array. DynamicTyping:True converts number string to actual numbers.

.reduce () singles a list of items to a final value. It starts with an initial value and adds down numbers to a final result. .reduce() is the right tool for summing and averging rows of data because it sums up arrays into one single list. It also groups array to an object by categories. 

 
### Chart Types
 
The bar chart is ideal for comparing absolute totals like sessions or revenue by source. Its vertical bars enable a side-by-side large volume comparisons and labels categories clearly.

The doughnut chart shows revenue share visualization by proportional segments of a whole. The hollow center and bottom legend improves readability for multiple sources and emphasizes relative contributions.

The line chart suits monthly trends by connecting data points over time, revealing patterns or seasonality across sources through smooth curves and chronological x-axis flow.

### Real-World Reflection
 
The story is that traffic sources are not equal: one or two channels drive most of the sessions while revenue is concentrated in the best-performing sources, so the Metro Report can see where audience attention and value are coming from. If one source consistently leads in sessions and revenue, it should be treated as the primary growth engine; if another source has high revenue but lower sessions, that channel deserves investment because it converts better. Overall, use the data to prioritize the channels that deliver the biggest audience and revenue impact.
